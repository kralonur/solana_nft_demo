# Solana NFT Demo

This project is nft demo project written for Solana. Programs are written in Rust language. Anchor framework is being used.
This project mints NFT for the user, when minting NFT fee should be sent to the contract. Fee is decided by the admin.
And the contract has maximum mint limit.
Token metadata is handled by metaplex.

Note: This project is not tested properly and is not production ready!
